# Interactive states

## Default state
- Hover - darken colour
- Focus - tintier highlight color outline
- Pressed - inner shadow

There are some exceptions to these rules, such as focus states of Text input and Choices.

## Buttons and Links

|   |  Normal |  Hover |  Focus |  Pressed |
|---|---|---|---|---|
|  Button |   |   |   |   |
|  Button Outline |   |   |   |   |
| Link  |   |   |   |   |
| Button Rounded  |   |   |   |   |
| Breadcrumb  |   |   |   |   |
